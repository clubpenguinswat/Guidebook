Server Tickets

Tickets can be created in SWAT's Discord server to ask for support. As a staff member, it is imperative to ensure that a healthy communication is maintained between the staff team as a whole and the person who requires assistance.

How It Works

A support ticket can be created by either using the $new command in any channel in the server, using the new slash command, or by interacting with the panel in the ❓】help-center channel. If a user interacts with the panel, they also get an option to state their request, issue, or question in a modal, and the information is displayed with the ticket introduction message.

General Guidelines

There are a few important notes to keep in mind…

  • Greet the requestor and appreciate them for reaching out.
  • Analyze their problem carefully, and list down all possible solutions for them.
  • Before marking a ticket as solved on your own, ask them if they have any other queries.
  • After a ticket has closed, give it a few hours, and then delete it.
  • Immediately before you delete the ticket, please save a transcript. This is important.

Game Nights

Some individuals create a support ticket with the intent to host a game night. In this case, please ask them to present what they are willing to play and what information should be shared publicly. You can ask them to answer in this format…

Hi! Please fill out the following details about your game night… - Name of the game - Link of the game - Duration of the gameplay - (Optional) Prize(s) at stake If your game is available on a gaming platform such as Roblox, please state that as well. If it is a game playable on a Minecraft server, please state that, too. Feel free to request anything else you may require! Additionally, you can also provide information on how to play, and so on, to give us a helping hand. Otherwise, we can help you fill out the details!

Giveaways

Some individuals create a support ticket with the intent to host a giveaway. In this case, please ask them to present what they are willing to give away and how. You can ask them to answer in this format…

Hi! Please fill out the following details about your giveaway… - Prize at stake - Number of winners - Duration of the giveaway - (Optional) Custom message Additionally, feel free to supply any other requirements, multipliers, blacklisted roles, and so on!

Giveaway Guidelines

There are a few important notes to keep in mind when hosting a giveaway…

  • Please use only the Cinnamon Discord bot for hosting giveaways.
  • The duration of the giveaway must be within the range of 1 hour and 2 weeks.
  • If SWAT Coins or Items are being given away, the prize must be sent to the SWAT Discord Bot first. Once sent, the giveaway can be set up.
  • Apply giveaway settings as requested. Do not apply any other giveaway settings to giveaways, unless otherwise requested to by the donor. This also means that visitors cannot be blacklisted unless asked by the donor.
  • Make sure to set the requestor as the giveaway donor in the giveaway settings.
  • Cinnamon Discord bot will ping the Giveaways role automatically upon hosting the giveaway. Do not accidentally ping twice.
