Server Tickets

Tickets can be created in SWAT's Discord server to ask for support. As a staff member, it is imperative to ensure that a healthy communication is maintained between the staff team as a whole and the person who requires assistance.

How it works

A support ticket can be created by either using the $new command in any channel in the server, or by interacting with the panel in the ❓】help-center channel. If a user interacts with the panel, they also get an option to state their request, issue, or question in a modal, and the information is displayed with the ticket introduction message.

Guidelines

  • Greet the requestor and appreciate them for reaching out.
  • Analyze their problem carefully, and list down all possible solutions for them.
  • Before marking a ticket as solved on your own, ask them if they have any other queries.
  • After a ticket has closed, give it a few hours, and then delete it.
  • Immediately before you delete the ticket, please save a transcript. This is important.
